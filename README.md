# buatmbggweh 🎁🎵

A high-fidelity, interactive, and beautifully designed romantic music gift web experience. This project features premium glassmorphism visuals, smooth realistic turntable/tonearm animations, functional audio controls, and an offline-first generative ambient synth engine using the Web Audio API.

## ✨ Features
- **Premium Design & Aesthetics**: Deep forest-green gradient theme with floating ambient bokeh light particles.
- **Glassmorphism Music Card**: Highly polished card interface with glowing active border reflections.
- **Realistic Turntable Mechanicals**: The tonearm sweeps 20 degrees onto the vinyl cover when playing, and the cover morphs into a spinning record with real-time glassy reflections.
- **Generative Lo-Fi Ambient Synth**: Procedural synthesis of soft ambient rhodes-piano chords (`Cmaj7`, `Am9`, `Fmaj7`, `G7/9`) and continuous vintage vinyl static crackle and rain sounds, working 100% offline.
- **Dynamic Seekbar & Full Playback Controls**: Draggable seekbar showing accurate elapsed time and interactive play, shuffle, and repeat modes.
- **Interactive Gift Customization**: Fully featured settings modal with inputs to personalize the gift headers, song details, and dedication texts.
- **Storage Persistence**: Saves all user kustomizations inside `localStorage` automatically.

## 📂 Project Structure
- `index.html` - Kerangka dasar website, panel pengaturan modal, dan ikon kontrol.
- `style.css` - Desain responsif, tata letak glassmorphism, dan seluruh animasi transisi.
- `app.js` - Logika pemutar, pensintesis audio, dan integrasi penyimpanan data.
- `assets/cover.png` - Aset gambar dasar bertema piringan hitam vintage dengan bokeh hangat.

## 🚀 Cara Menjalankan Halaman
1. Kloning repositori ini atau unduh kodenya.
2. Klik ganda file `index.html` pada browser modern apa saja.
3. Tekan **Play** untuk memulai piringan hitam dan mendengar melodi piano yang menenangkan!
